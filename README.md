A showcase of Jetpack Compose animation APIs brought together in one project.  
**Animation Hub** demonstrates practical usage of value animations, transitions, visibility, gestures, and more, all in a single hub for developers to learn and explore.

- Jetpack Compose for UI
- Kotlin Coroutines for async animation handling
- Material Design 3
- MVVM-inspired modular structure
- Compose Animation APIs:
  - `animate*AsState`
  - `Animatable`
  - `updateTransition`
  - `AnimatedVisibility`
  - `AnimatedContent`
  - `animateContentSize`
  - `animateItemPlacement`
  - `rememberInfiniteTransition`
  - `graphicsLayer`

## ‚öôÔ∏è Included Animations

- **Value Based** ‚Üí Animate size, color, and corners with `animate*AsState`.  
- **Transition** ‚Üí Animate multiple properties (size, radius, color) together using `updateTransition`.  
- **Visibility** ‚Üí Enter/exit animations with `AnimatedVisibility` + `Crossfade`.  
- **Animatable** ‚Üí Gesture-based movement, springs, and flings with `Animatable`.  
- **Animated Content** ‚Üí State-based content swap with animated transitions.  
- **Animate Content Size** ‚Üí Smoothly expand/collapse layouts when content changes.  
- **Animate Item Placement** ‚Üí Animated list reordering with `animateItemPlacement`.  
- **Draggable** ‚Üí Drag and snap back with spring physics.  
- **Graphics Layer** ‚Üí Transformations (rotation, scale, alpha) via `graphicsLayer`.
